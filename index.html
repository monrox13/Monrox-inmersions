¿Qué es Monrox Inmersions?

Un proyecto experimental que combina neurotecnología, sensores de cuerpo completo y realidad virtual para explorar experiencias más allá de las capacidades humanas.

Tecnología Utilizada

  • BCI: Uso de sistemas como OpenBCI Galea para leer señales cerebrales.
  • Motion Capture: Trajes con sensores para capturar movimiento corporal completo.
  • VR Engine: Unreal Engine para construir el entorno inmersivo.
